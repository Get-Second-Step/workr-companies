# workr-companies

Default agent personas and instruction files for [Workr](https://workr.space) — the AI agent workspace for performance marketers.

## Structure

```
default/
  ceo/
    AGENTS.md      — System prompt / instruction file for the CEO agent
    HEARTBEAT.md   — Checklist the CEO runs on every scheduled heartbeat
    SOUL.md        — CEO persona: strategic posture, voice, and tone
    TOOLS.md       — Tools the CEO has access to (add as you go)
```

## Usage

When onboarding a new workspace in Workr, the CEO agent is bootstrapped with this persona. Point the agent's instruction file to `default/ceo/AGENTS.md`.
